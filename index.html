Research Notes

  • Off Policy Monte Carlo Control Application in Racetrack

    The study demonstrates how off-policy Monte Carlo control can learn an optimal racetrack policy and compares it with Monte Carlo methods in Blackjack.

  • Color Calibration in Machine Learning

    This study integrates matrix transformation and non-linear mapping techniques to develop and validate a robust color calibration model that achieves precise color reproduction in a real-time MLOps system.

  • Evaluating Q-Estimates in Nonstationary Bandit Environments

    This study shows that the constant step-size method (α = 0.1) outperforms the sample-average method in nonstationary bandit problems by adapting faster to changing rewards, leading to higher performance.

  • Explore Best ε for 10-armed Bandit Problems

    Set up a ten-armed bandit experiment and explore how different ε behave.

  • Centroid Based Line Following

    Extract colors in HSV format and mark the centroid to achieve single color line following.

  • A Simple Maze Escape Solution

    Use the simplest logic and control to get out of the maze.
